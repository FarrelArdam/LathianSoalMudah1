Fungsi:

Program ini digunakan untuk menghitung total biaya pengiriman paket berdasarkan:

Berat paket (kg)

Jarak tempuh (km)

Volume paket (panjang × lebar × tinggi dalam cm³)

Cara Kerja:

Input data:
Pengguna memasukkan berat, jarak, panjang, lebar, dan tinggi paket melalui Scanner.

Hitung volume:
Volume dihitung dengan rumus volume = panjang * lebar * tinggi.

Tentukan biaya per kg:

Jika jarak ≤ 10 km → biaya per kg = Rp 4.250

Jika jarak > 10 km → biaya per kg = Rp 6.000

Hitung total biaya:
totalBiaya = berat * biayaPerKg

Tambahkan biaya volume (jika perlu):
Jika volume > 100 cm³, maka ditambah biaya Rp 50.000.

Tampilkan hasil:
Program menampilkan semua data dan total biaya pengiriman.
